# Python Practice Repository

This repository contains multiple Python practice tasks covering different topics.

## Practice2
Covers:
- Booleans
- If-Else statements
- Loops

## Practice-03
Covers:
- Functions
- Lambda expressions
- Classes
- Inheritance

## Practice4
Covers advanced Python concepts:
- Iterators and Generators
- Date and Time (datetime module)
- Math and Random modules
- JSON parsing and file operations

Each topic is separated into different Python files for better structure and readability.

All examples are based on W3Schools Python tutorials.
